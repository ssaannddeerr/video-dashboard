# Screenshots

## How to Add Screenshots

1. Run the application:
   ```bash
   npm start
   ```

2. Take screenshots using your system's screenshot tool:
   - **macOS**: Press `Cmd + Shift + 4` then Space to capture a window
   - **Windows**: Press `Win + Shift + S` or use Snipping Tool
   - **Linux**: Use your distribution's screenshot tool (usually `PrtScn`)

3. Recommended screenshots to capture:
   - `dashboard.png` - Full view of the 3x4 video grid
   - `hover-effect.png` - Video container with hover effect visible
   - `placeholder.png` - View showing placeholder slots

4. Save the screenshots in this directory with the filenames mentioned above
